26 Commands | () - Optional | <> - Required.
-------------------------------------------------------
>>activity | Lists all server activity in the form of messages.
>>activity <@mention> | Shows the number of messages sent by a specific user.
>>admins | List server administrators or Set Administrator roles.
>>aliases | List number of aliases for each command category
>>aliases <command> | List all aliases for the specific command.
>>avatar | Shows your avatar.
>>avatar <@mention> | Shows the mentioned person(s) avatar.
>>banner | Shows your banner.
>>banner <@mention> | shows the mentioned person(s) banner.
>>botinfo | Shows information about the bot.
>>channelinfo | Shows information about the current channel.
>>channelinfo <#channel> | Shows info about the current channel.
>>emojis | List all emojis in the current server/guild.
>>help | Shows help on the bot.
>>inviteme | Shows a message with the invite link for the bot.
>>mods | List server Moderators or set Moderator roles.
>>permissions | List's bot permissions.
>>permissions <@role> | Shows permissions for the mentioned role.
>>ping | Shows bot's latency/ping.
>>prefix | Shoes the prefix for the bot.
>>roleinfo <@role> | Shows info about the mentioned role.
>>servercount | Shows how many Servers and Users are using NextGen.
>>servericon | Shows the servers/guilds icon.
>>serverinfo | Shows info about the current server/guild.
>>serverstaff | Shows list of people who are Admins and Mods.
>>stats | Shows bot stats.
>>uptime | Shows the bots uptime.
>>vote | Shows Vote link for the bot.
>>whois <@mention> | Show User info on mentions person(s).
>>modactivity | Show activity of moderators and higher.
